# هتل اصفهان — سیستم رزرواسیون آنلاین

یک وب‌اپلیکیشن کامل رزرواسیون هتل، طراحی‌شده برای هتل‌های سنتی-مدرن ایرانی. این پروژه با فناوری‌های پایه وب ساخته شده و هیچ وابستگی به فریم‌ورک خارجی ندارد.

---

## معرفی

این پروژه یک نمونه واقعی از سیستم رزرواسیون هتل است که با HTML، CSS و JavaScript خالص پیاده‌سازی شده. هدف اصلی طراحی تجربه‌ای روان و حرفه‌ای برای مهمانانی است که قصد رزرو اتاق در هتل‌های اصفهان را دارند — از انتخاب اتاق تا تأیید نهایی، همه چیز در مرورگر اتفاق می‌افتد.

---

## صفحات

- **صفحه اصلی** — معرفی هتل، اتاق‌های محبوب و نظرات مهمانان
- **لیست اتاق‌ها** — فیلتر بر اساس نوع، قیمت و ظرفیت به همراه جستجوی زنده
- **جزئیات اتاق** — گالری تصاویر، امکانات و فرم رزرو با محاسبه خودکار قیمت
- **رزروهای من** — مدیریت و لغو رزروهای ثبت‌شده

---

## ویژگی‌ها

- **کاملاً آفلاین** — بدون نیاز به اینترنت، سرور یا دیتابیس
- **تقویم شمسی** — نمایش تاریخ‌های ورود و خروج به فارسی
- **محاسبه خودکار** — تعداد شب‌ها، مالیات ۱۰٪ و قیمت نهایی
- **localStorage** — رزروها در مرورگر ذخیره و قابل مدیریت هستند
- **طراحی ریسپانسیو** — سازگار با موبایل، تبلت و دسکتاپ
- **فونت لوکال** — فونت وزیرمتن بدون وابستگی به CDN
